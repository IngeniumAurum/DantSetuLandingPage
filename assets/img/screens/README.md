# App screenshot (showcase section)

The "See it in action" section loads one screenshot into the browser mockup:

| File          | Frame          | Suggested size / ratio |
| ------------- | -------------- | ---------------------- |
| `app-web.png` | Browser window | ~1600×1000 (16:10)     |

Drop the screenshot here with that exact filename. The frame shows a labelled
placeholder until the file exists, then displays the screenshot automatically —
no code change needed.

The image is fitted with `object-fit: contain` inside a padded frame, so **any
size works**: the whole screenshot is shown (never cropped), and a mismatched
ratio simply gets clean spacing around it.
